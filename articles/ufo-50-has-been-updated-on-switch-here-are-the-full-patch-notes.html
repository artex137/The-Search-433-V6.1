UFO 50 Has Been Updated on Switch

UFO 50, a retro compilation for the Nintendo Switch, has received an update to version 1.7.8.30, according to a report by Nintendo Life. The update includes several enhancements, such as a new 'pixel perfect' scaling option and an updated home menu icon. No injuries or fatalities have been reported in connection with this update.

This update is significant for gamers, as it improves the user experience and addresses previous issues with the game's performance. Officials from the development team have stated that more updates will follow in the future.

Key facts

  • Update version: 1.7.8.30
  • New features include: pixel perfect scaling, button swapping, and an updated icon
  • Optimizations to address performance slowdown
  • Issue with copying Empty Profile has been addressed
  • Global language setting now saves properly

Timeline

UFO 50 was shadow-dropped last month and received a positive reception, scoring 9/10 from Nintendo Life. The latest update was released recently, with the patch notes detailing the improvements.

Following the update, players have reported mixed experiences, with some noting ongoing issues that remain unaddressed.

Official statements

“We were absolutely enamoured with UFO 50 when it shadow-dropped last month,” said the team at Nintendo Life. “This update is a good start, and we look forward to what is next.”

Context

UFO 50 has been well-received since its launch, praised for its diverse game collection. However, some players have expressed dissatisfaction with the initial icon design and ongoing game issues. The development team has acknowledged these concerns and is working on further updates.
