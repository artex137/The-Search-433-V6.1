UFO 50 update out now on Nintendo Switch, patch notes

Published 2025-09-09 13:50 UTC

UFO 50 update out now on Nintendo Switch

A new update for the Nintendo Switch version of UFO 50 has been released, with version 1.7.8.30 now available for download. The update includes various enhancements based on user feedback but does not report any injuries or fatalities.

This update is significant as it addresses several performance issues and user interface improvements, which aim to enhance the gaming experience. The developers have responded to player feedback, indicating a commitment to ongoing support for the game.

Key facts

  • Update version: 1.7.8.30
  • Release date: Not disclosed
  • Changes: Home menu icon updated, optimizations for slowdown, pixel perfect scaling option added, button swapping feature added, crash issue addressed, global language setting now saves properly
  • Purchase: Available as a digital download on Nintendo Switch
  • Physical release: Planned for a later date

Timeline

The update for UFO 50 has gone live, responding to player feedback regarding performance issues and interface adjustments. Specific enhancements include the addition of a pixel perfect scaling option and a new button swapping feature.

The developers have also optimized the game to address slowdown, with plans for further improvements in future updates.

Official statements

“Following feedback, the home menu icon has been updated and various optimizations have been implemented,” the developers stated.

Context

The release of UFO 50 has garnered attention for its retro-style gameplay and unique features. Player feedback has been instrumental in shaping the updates, demonstrating the developers' responsiveness to their community.

What’s next

The developers plan to continue enhancing UFO 50 with further updates aimed at improving game performance and user experience. The exact timeline for future updates remains not disclosed.
